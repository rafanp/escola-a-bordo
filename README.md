## 🚀 Como executar o projeto

### Pré-requisitos

Ambiente React configurado

#### 🎲 Rodando o projeto

```bash

# Clone este repositório
$ git clone https://github.com/rafanp/GoBarber

# Instale as dependências
$ yarn

# Execute a aplicação front end
$ yarn react-native run-android

```
